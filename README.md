#  BarBar
A hairstyle recommender using face shape
 
 # How to Start?

## Setup Locally

- Clone the repo: 
    ```
    git clone https://github.com/PARKER-X/BarBar/
     ```
    
After cloning successfully

- Cmd to Run: 
    ```
    python manage.py runserver
